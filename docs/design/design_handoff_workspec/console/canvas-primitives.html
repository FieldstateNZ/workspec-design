Canvas primitives.

The atoms of WorkSpec. Every screen is built from these. Defined as classes in shell.css so the same markup works in static mockups and in the live prototype.

01Artifact card

The fundamental unit. Type chip on top, title bold, optional sub. Type color signals kind. .selected adds accent border + glow. .badge in the corner for version pills.
