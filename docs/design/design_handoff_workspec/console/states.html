02Loading

Three loading vocabularies. Spinner is for actions. Skeleton is for content. Streaming dots are for the agent — never for static loading.

Spinner

Action in flight: saving, navigating, opening a modal. Sub-1s actions don't show a spinner — they just complete.

Skeleton
atlas thinking

Streaming

The agent is working. Pulse + dots. Always show the user how to interrupt: Esc.

03Errors

Errors are owned. We say what happened, what we tried, what to do next. Never "something went wrong."
