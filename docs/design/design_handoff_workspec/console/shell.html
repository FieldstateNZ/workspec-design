02Region notes

Top bar · 40px

Breadcrumb path on the left. Lens toggle just right of breadcrumbs (it's a primary action, not a setting). Command palette trigger, agent status, and avatar on the far right. Always present, never collapses.

Left rail · 64px

Icon-only navigation. Five primary destinations (canvas, containers, boards, history) plus search and settings pinned to the bottom. Hover for tooltip. Active state uses --accent color, no fill.

Canvas · flex

The work surface. Pannable, zoomable, infinite. Grid background at 24px snaps to the spacing scale. Artifacts, edges, bubbles, and lassos all live here. Bubbles anchor to canvas points, not viewport — they pan with the canvas.

Right detail · 320px

Context for the current selection. When nothing is selected, shows a session log (recent agent moves). Collapses to 0 when the user hits ⌘\. Width is fixed — content reflows below itself, never compresses.

Responsive behavior

Below 1280px, detail panel auto-collapses (still toggleable). Below 960px, left rail collapses to bottom dock. WorkSpec is desktop-first; mobile is read-only — out of scope for v1.
