03 · Principles

Three rules of the room

01

Every artifact has a hand

Drawn by, dated, drafted-or-final. Versions stay legible. The user should be able to glance at any object and know who put it there, when, and how confident the system is.

02

The agent draws in pencil

Atlas's contributions appear as ghosted pencil — visible, undoable, attributable. They solidify into ink only when accepted. AI is a draftsman, not a daemon.

03

Slow over fast

This is a contemplative tool. Generous margins, deliberate type, no urgency theater. Paper invites the user to think before they ship.
