ADR-003 ACCEPTED · decided by Brett Smith · 12 Jun 2026

GitHub is the source of truth for published content

LINKED API Application Publish Worker GitHub · supersedes ADR-001
Context

Quire is a git-native CMS. We considered rendering on request from the database, but authors expect their site to be a static, versioned artifact they can inspect and roll back. Content already round-trips through branches and pull requests.

Decision

Published content lives in the author's GitHub repository as the canonical store. The Publish Worker builds and commits; the live site is served from git. Postgres holds drafts and operational state only.

Consequences

Rollback and audit come free from git history. The publish path now depends on GitHub availability — see the linked risk on the publish journey. ADR-001 (render at request time) is superseded.
