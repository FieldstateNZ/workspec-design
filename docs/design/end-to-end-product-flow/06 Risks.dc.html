LINKED GitHub Publish Worker Journey · Publish ADR-003
Description

ADR-003 makes GitHub the source of truth for published content. If GitHub is unavailable, the Publish Worker cannot commit and the live site cannot update — every author is blocked, with no degraded mode.

Mitigation

Queue publish jobs durably and retry with backoff; surface a clear "publishing paused — GitHub unavailable" state to authors rather than a silent failure. Owner to spike a queue-and-retry slice in the next build cycle.
