Quire CMS

v0.4 · git-native CMS & blogging platform · compiled from 6 design surfaces

1 · Overview

from Discovery

Quire is a git-native CMS. Authors write posts in Quire; content publishes to a GitHub repository through branches, pull requests, and build/deploy actions. The live site is served from git.

2 · Personas & journeys

from Journeys

The Author moves a post through Draft → Review → Publish → Live. The publish stage is the project's known risk zone — see §6.

3 · Architecture

from Architecture

A Web App and API Application front a PostgreSQL store; a Publish Worker builds and commits to GitHub. External systems: GitHub, Claude, Stripe, Sendgrid.
