# Drift log

Every discrepancy found while extracting the design system, logged for Brett's adjudication —
**nothing here has been resolved unilaterally.** Extraction moves token names, values,
selectors, and component behaviour exactly as they exist in `workspec/artifacts/workspec`;
where the handoffs and the implementation disagree, or the implementation is internally
inconsistent, the disagreement is recorded, not fixed.

"Handoff" cells cite `docs/design/<bundle>/...` (vendored verbatim in this repo);
"Implementation" cells cite `workspec/artifacts/workspec/...` (not vendored here — the live app,
read during extraction).

"Migrated-to mapping"), found by `packages/design/scripts/verify-migration.ts`
(`pnpm --filter @workspec/design run verify:migration`), which Prettier-normalizes and
import-strips every migrated file against its enterprise source and diffs the rest. 70 of 75
